# bw-cli [![Build Status](https://travis-ci.org/Bedework/bw-cli.svg)](https://travis-ci.org/Bedework/bw-cli)

This project provides a command line client for
[Bedework](https://www.apereo.org/projects/bedework).

## Requirements

1. JDK 21
2. Maven 3
